# SAAM Portfolio Project

Project structure:
- notebooks/: notebooks for cleaning and analysis
- src/: helper scripts
- data_raw/: raw input files (not tracked)
- data_clean/: cleaned outputs (not tracked)

Setup:
1. Clone the repo
2. Create a virtual environment
3. Install dependencies with `pip install -r requirements.txt`

Note:
Raw data files are not included in the repository. Each group member should place the Excel files inside `data_raw/`.
